Click the Source menu to choose between Resource file or Web service.

Resource file is a local database. This file can be used without internet.

Web service is provided by www.inspired.jsu.edu. Therefore, you must have internet to access this service.

Click on Course to view a dropdown menu of courses.

Click on Column to view a dropdown menu of assignments.

 

Click on a bar to view the coordinating student’s information.

The information is listed below the Leaderboard and includes the student’s ID, Name, Email, and Score.

Term and Enrollment will update when you choose a Course.
